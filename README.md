Features

Dashboard
- Real-time business overview
- Key metrics and statistics
- Recent orders tracking
- Revenue monitoring

Cake Management
- Add new cake products
- Edit existing cake details
- Delete products from menu
- Categorize cakes (Birthday, Wedding, Anniversary, Custom)

 Order Management
- Create new customer orders
- Track order status (Pending, Preparing, Completed)
- Manage delivery dates
- Order history and tracking

 Inventory Management
- Stock level monitoring
- Low stock alerts
- Supplier information
- Stock updates and adjustments

 Advanced Features
- Search and filter functionality
- Responsive mobile design
- Local storage data persistence
- Professional user interface

 Technologies Used

- Frontend: HTML5, CSS3, JavaScript (ES6+)
- Storage: Browser Local Storage API
- Icons: Font Awesome 6.4.0
- Styling: CSS Grid & Flexbox
- Design: Responsive Web Design
- Deployment: GitHub Pages


No installation or server required - everything runs locally in your browser.
